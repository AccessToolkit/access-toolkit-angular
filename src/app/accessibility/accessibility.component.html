Introduction

This site was made with accessibility in mind and with the intention that everyone should be able to use it. The site should meet the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA, however it is possible that issues and barriers to use were missed as we are a small team with limited resources. There are also accessibility and usability features that are missing from the site that we have listed below.

Notes on known issues

  • The site is missing British Sign Language and Irish Sign Language versions.
  • The site is missing an "easy read" version.
  • While this page should work correctly with a screen reader, it is missing the audio versions of the text recorded for the other pages. This is both because it was written later, and also because this page will change in response to discovering and fixing issues.
  • Colour mode picker and navigation menu are only navigable with the tab button and not with arrow keys as in best practice.
  • Colour mode picker doesn't hide itself when it loses keyboard focus.
  • The site uses the browser audio player for maximum compatibility, but this may cause issues for users. See Browser Video Players Review by Adrian Roselli for a recent comparison of the browser media players.
  • The site was mostly tested with the Chrome browser and NVDA screen reader. There may be issues when using other screen reader and browser combinations.

Accessibility features

  • "Skip to content link" as well as quick link to this page for keyboard users
  • Colour mode picker: All themes pass contrast requirements, but some users prefer darker colours, and contrast for some themes may not be sufficient for colour-blind users.
  • Resizable text
  • Table of contents for easier navigation through big pages, as well as a scroll to top link so keyboard users are not stuck at the end of a page.
  • Audio recordings of our main page contents that can be downloaded and listened to away from the site.

How we work towards site accessibility

The Access Toolkit for Art Workers site has been designed and tested by Saerlaith Uaid Ní Dhuibhir to comply with the World Wide Web Consortium's Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA, the level legally required in many countries. Where possible, we have aimed to meet Level AAA requirements also.

User testing beyond simply meeting guidelines is a vital part of making a website accessible. However, this has been beyond the scope of this small project. If there is any part of this website that remains inaccessible for you, please contact Iarlaith Ní Fheorais at iarlaithnifheorais"@"gmail.com
